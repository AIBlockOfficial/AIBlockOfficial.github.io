Valence

The AIBlock L2 node for data exchange between peers. Complete with E2E encryption.

About the project‚Äã

Valence nodes are a layer 2 solution for massively improving the scaling, throughput and efficiency of the AIBlock network. They can process transactions, rollups, 2-Way transactions and even mint and facilitate the trade of items. They're also completely distributed, which means that anybody can spin a Valence node up and get started, right away.

The core Valence node code can be found on GitHub here, but what really makes Valence nodes scale the AIBlock network functionality is plugins. Plugins extend the functionality of your Valence node and give it new abilities to handle specific use cases. Examples of Valence node plugins are:

  • Core, which provides reusable functionality for a variety of other Valence node plugins
  • Market, which turns your Valence node into a marketplace within seconds

The Valence node code is open source, so you're welcome to contribute with your own plugins, or use plugins built by other members of the AIBlock community.
